# Section page headers

Full-width strip at the top of catalog pages (**200 px** tall on screen). Images use `background-size: cover`.

| Section | File path |
|---------|-----------|
| Loot Collector | `public/images/headers/loot-collector-header.webp` |
| Armor sets | `public/images/headers/armor-sets-header.webp` |

| | |
|---|---|
| **Recommended size** | **2400 × 400 px** (6 : 1) |
| **Minimum** | **1200 × 200 px** |

**Design tips:**

- Keep the **bottom-left** area readable for the title (dark or uncluttered).
- Important art in the **center/right**; edges may crop on mobile.
- Export as WebP or JPG, sRGB.

Home banners for the same sections live in `public/images/banners/`. Activity raid/dungeon headers use `public/images/rad-loot/headers/`.
