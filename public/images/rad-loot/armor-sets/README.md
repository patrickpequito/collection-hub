# Activity armor set previews

One preview image per activity, showing all three class sets together. Displayed below the piece icons in the **Armor sets** panel on each `/rad-loot/[slug]` page.

## Path

`public/images/rad-loot/armor-sets/{activity-slug}.webp`

## Filename pattern

Use the same slug as the activity page. **One file per raid or dungeon.**

| Example file | Activity |
|--------------|----------|
| `vault-of-glass.webp` | Vault of Glass |
| `prophecy.webp` | Prophecy |

## Format

- **WebP** (preferred) or JPG, sRGB
- **Recommended width:** **1200 px** wide (Hunter, Titan, and Warlock side by side); height follows your composition
- The preview scales to the panel width; click to open a larger popup view
