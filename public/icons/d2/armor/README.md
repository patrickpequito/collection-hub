# Armor slot icons

Icons for armor **slot type** (Helmet, Gauntlets, etc.) on armor detail pages.

## Path

`public/icons/d2/armor/{filename}.png`

## Files

| File | Slot label |
|------|------------|
| `helmet.png` | Helmet |
| `gauntlets.png` | Gauntlets |
| `chest.png` | Chest Armor |
| `legs.png` | Leg Armor |
| `class_item.png` | Class Item |

## Format

- **PNG**, transparent background
- Square source art works well (~64–128 px); renders at **16 × 16 px** next to the slot label
